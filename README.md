# SuiGen - AI X NFT

## What is SuiGen?
SuiGen is the first prompt artist platform on Sui ecosystem with artist creativity at its heart. The platform lets you mint your imagination into an AI-Generated NFT which is verified by an immutable blockchain ledger. Collectors and enthusiasts can buy and sell the NFT together with the prompt that was used to generate the image.  Additionally, users can explore and purchase NFTs created by other talented artists, expanding their digital art collection.

## Technologies Used
- [Sui blockchain](https://sui.io/)
- [Stable diffusion image model](https://huggingface.co/)
- [React](https://reactjs.org/)
- [Next.js](https://nextjs.org/)
- [huggingFace](https://huggingface.co/)
