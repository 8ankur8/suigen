# SuiGen - AI X NFT

## What is SuiGen?
SuiGen is the first prompt artist platform on Sui ecosystem with artist creativity at its heart. The platform lets you mint your imagination into an AI-Generated NFT which is verified by an immutable blockchain ledger. Collectors and enthusiasts can buy and sell the NFT together with the prompt that was used to generate the image. It's important to note that similar prompts will never produce the same art to highlight the beauty of algorithm randomness.

## Technologies Used
